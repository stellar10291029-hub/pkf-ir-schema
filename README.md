# pkf-ir-schema

Reference implementation artifacts for an AI-facing Intermediate Representation (IR) schema of the Pain Kernel Framework (PKF).

The authoritative paper is archived on Zenodo (https://zenodo.org/records/18012858).

## What this repo provides
- A machine-readable IR schema (JSON Schema + YAML).
- Minimal examples for integration and testing.

## What this repo does NOT provide
- Full paper text, abstract, or exhaustive theory.
- Clinical advice, prediction, or optimization.
- A required implementation; this is a reference spec only.

## Files
- `schema/pkf-ir.schema.json` — JSON Schema (authoritative within this repo)
- `schema/pkf-ir.schema.yaml` — YAML mirror
- `examples/` — minimal examples
